## Welcome to TernSecure

TernSecure provides a robust and easy-to-implement Firebase authentication solution for Next.js 16+. Built to leverage Node.js-based proxies, TernSecure seamlessly integrates with the Firebase Admin SDK for secure, server-side authentication.

## Key Features

- **Node.js Proxy Support**: Leverages Next.js 16+ Node.js-based proxies for server-side authentication
- **Firebase Integration**: Built on top of Firebase Auth with Admin SDK support
- **Type Safety**: Full TypeScript support with comprehensive type definitions
- **Client & Server**: Seamless authentication state access on both client and server
- **Route Protection**: Easy-to-configure route protection with proxy-based middleware
- **Multiple Providers**: Support for Google, Email/Password, and other Firebase auth providers

## Quick Installation

Get started with TernSecure in seconds:

```bash
pnpm add @tern-secure-node/nextjs firebase firebase-admin
```

Ready to dive in? Start with the [Next.js Guide](/docs/nextjs-guide) to implement authentication in your application.
